CSS Structure

There are 5 main CSS files in this theme.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.


Open The css Folder, Find All CSS Files.

  • animate.css - Animation Style
  • bootstrap.min.css - Bootstrap Style
  • jquery.fancybox.min - Fancy Box Style
  • style.css - Theme Common Styles With Responsive Style In It
  • swiper-bundle.min.css - Swiper Slider Style

Javascript Structure

This theme imports 12 Javascript files. All located in the "js" folder.

Open The js Folder, Find All JAVA Script Files.

  • bg-moving.js - Moving JS
  • bmi-calculator.js - BMI Calculator JS
  • bootstrap.min.js - Bootstrap JS
  • coming-soon.js - Coming Soon JS
  • custom.js - Main Custom JS
  • custom-scroll-count.js - Scroll Count JS
  • font-awesome.min.js - Font Awesome Icon JS
  • jquery.fancybox.min.js - Fancy Box JS
  • jquery-3.6.0.min.js - JQuery JS
  • popper.min.js - Bootstrap JS
  • swiper-bundle.min.js - Swiper Slider JS
  • wow.min.js - WOW Animation JS
